# AI Talking Points

## What is Groq?
Think of Groq as the ultra-fast engine powering our AI. Unlike traditional AI providers that run on standard graphics cards, Groq uses specialized chips designed specifically for AI generation, making responses nearly instantaneous.

## What are Prompts?
A prompt is simply the instruction manual we give the AI. Behind the scenes, when a user types "Reduce carbon," our system wraps that into a detailed command: "You are an ESG expert. Give 3 actionable corporate policies for 'Reduce carbon' in a strict JSON format." This ensures the AI stays on topic and formats its answers perfectly every time.

## Security Highlights
- **Prompt Injection Defense:** We actively scan for "jailbreak" attempts to ensure the AI isn't tricked into revealing secrets or acting maliciously.
- **Data Privacy:** No user IDs or personal data ever leave our servers.
- **Access Control:** Only our verified Java Backend can talk to the AI service, thanks to JWT tokens.
