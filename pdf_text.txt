9. COMMON MISTAKES — Read This Now
These 8 mistakes are made every sprint. Avoiding them saves days of wasted work.
 Mistake
 Why It Happens and How to Avoid It
[X] Hardcoding secrets in
config files
Use ${ENV_VAR} in application.yml and os.getenv() in Python. Actual values go only in
.env (which is in .gitignore). If you commit a secret to GitHub, rotate it immediately —
deleting the file is not enough.
[X] Committing .env to
GitHub
Add .env to .gitignore on Day 1 before any commit. Run git status before every git
commit to verify. GitHub scans for exposed keys and will warn you, but by then it is
already public.
[X] Not wrapping Groq calls
in try-except
The Groq free tier has rate limits and occasional outages. Every call must be in a
try-except block. On failure: log the error, return the fallback template with {is_fallback:
true}. Never return HTTP 500 because the AI is unavailable.
[X] Not setting up
docker-compose until Week
3
Set it up on Day 5. The most common Demo Day failure is 'works on my laptop but not
the demo machine.' Test the full docker-compose stack every day from Day 5 onwards.
Discover integration problems early.
[X] Adding features in Week
4
Week 4 is for demo preparation only. Feature freeze after Day 15. Every hour building
features in Week 4 is an hour not spent making the demo flawless. New ideas go in
GitHub Issues labelled post-sprint.
[X] Empty database on
Demo Day
Implement the DataLoader seeder on Day 14. On Demo Day: docker-compose down -v
then docker-compose up to reset to a clean seeded state. An empty list on Demo Day
undermines the entire presentation.
[X] Not reading Groq docs
before writing AI code
Spend 30 minutes at console.groq.com/docs before Day 2. Key parameters: model
name, temperature (0.3 for factual, 0.7 for creative), max_tokens, and the messages
array structure. Misunderstanding temperature alone can waste an entire day.
[X] Skipping the demo dry
run
Every team that skips the rehearsal on Day 17 regrets it. Problems are always found in
rehearsal that you never noticed before. Do two full rehearsals (Day 17 and Day 19).
Use a stopwatch. The time limit is firm.
